# 📚 AI-Powered Personalized Study Planner

A modern Streamlit app for BTech/degree students to create a personalized day-by-day study schedule based on their syllabus and exam date.

## ✅ Features
- Upload syllabus as **PDF, image (JPG/PNG), or plain text**
- AI extracts every topic automatically
- Tick topics you've already completed
- Get a day-by-day plan from today → exam date
- Download plan as a **styled PDF**
- Works **without any API key** (built-in rule-based planner)
- **No Tesseract needed** — uses EasyOCR (pure Python)

## 🚀 Setup (Windows / PowerShell)

### 1. Go to the project folder
```powershell
cd "C:\Users\devay\Desktop\study-planner"
```

### 2. Create virtual environment
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 3. Install dependencies
```powershell
pip install -r requirements.txt
```
> ⚠️ First run will download EasyOCR models (~100MB). This is one-time only.

### 4. (Optional) Add Gemini API key for smarter extraction
```powershell
setx GEMINI_API_KEY "your_key_here"
```
Or just paste it in the sidebar when the app runs.

### 5. Run the app
```powershell
.\.venv\Scripts\python.exe -m streamlit run app.py
```

## 📱 How to use
1. **Step 1** — Upload your syllabus PDF/image or paste text
2. **Step 2** — Click "Extract all topics"
3. **Step 3** — Tick the topics you've already studied
4. **Step 4** — Set your exam date → Generate plan
5. **Step 5** — View your plan and download as PDF

## 🗂️ File Structure
```

├── app.py           # Main Streamlit UI

├── topic_parser.py  # Topic extraction (Gemini + heuristic)
├── planner.py       # Day-by-day plan generator

```

## ☁️ Deploy to Streamlit Cloud
1. Push to GitHub
2. Go to share.streamlit.io
3. Select repo, set main file as `app.py`
4. Add `GEMINI_API_KEY` in Secrets (optional)
5. Deploy!
